Game Shop by Nikolay Todorov. Java Console Application.
This is application which simulates online game shop.
There are users with three different roles - customer, seller, admin.
If you register the customer role will be automatically assigned to you.
Every user can edit their own data.
Customers can browse and buy games and deposit money.
Sellers can publish and edit games and recieve money on every game sold.
Admins can manage users, categories and games.
